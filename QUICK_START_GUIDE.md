## ✨ 交互特性

### 登录页面
1. **输入框聚焦**: 边框变紫色加粗
2. **密码显示/隐藏**: 点击眼睛图标切换
3. **实时验证**: 输入时即时提示错误
4. **加载状态**: 登录时显示进度圈
5. **成功反馈**: 绿色 SnackBar 提示
6. **错误反馈**: 红色 SnackBar 提示

### 注册页面
1. **必填标记**: 字段后显示 * 号
2. **密码确认**: 必须输入两次且一致
3. **可选字段**: 邮箱和手机号可选
4. **密码强度**: 实时验证密码格式
5. **提交按钮**: 加载时禁用防止重复提交
6. **自动返回**: 注册成功后自动跳回登录页

## 🎯 验证规则提示

### 用户ID
- ✅ 必填
- ✅ 长度: 2-20 字符
- ✅ 格式: 字母 + 数字 + 特殊字符(_@#$%^&*!)
- ❌ 示例: `u`, `user@123`, `张三李四王五赵六孙七周八吴九郑`

### 用户名
- ✅ 必填
- ✅ 长度: 2-20 字符
- ❌ 示例: `张`, `这是一个非常长的用户名测试`

### 密码
- ✅ 必填
- ✅ 长度: 8-20 字符
- ✅ 必须包含: 字母 + 数字
- ✅ 允许字符: 字母、数字、_@#$%^&*!
- ❌ 示例: `12345678` (无字母), `Abc@12` (太短), `Abc@12345678901234567890` (太长)

### 确认密码
- ✅ 必填
- ✅ 必须与密码完全一致
- ❌ 示例: 与密码字段不同

### 邮箱 (可选)
- ✅ 标准邮箱格式
- ✅ 为空时跳过验证
- ❌ 示例: `abc`, `@example.com`, `abc@`

### 手机号 (可选)
- ✅ 中国大陆11位手机号
- ✅ 以1开头，第二位3-9
- ✅ 为空时跳过验证
- ❌ 示例: `12345678901`, `21234567890`

## 🚀 快速开始

### 1. 配置 API 地址
编辑 `lib/services/api_service.dart`:
```dart
static const String baseUrl = 'http://你的API地址:端口';
```

### 2. 运行应用
```bash
flutter pub get
flutter run
```

### 3. 测试流程
1. **启动应用** → 看到登录页面
2. **点击"立即注册"** → 进入注册页面
3. **填写注册表单**:
   - 用户ID: `testuser123`
   - 用户名: `测试用户`
   - 密码: `Test@1234`
   - 确认密码: `Test@1234`
   - 邮箱: `test@example.com` (可选)
   - 手机号: `13800138000` (可选)
4. **点击"注册"** → 等待响应
5. **注册成功** → 自动返回登录页
6. **输入账号密码** → 点击"登录"
7. **登录成功** → Token 已保存
