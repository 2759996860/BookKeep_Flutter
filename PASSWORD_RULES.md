# 密码验证规则说明

## ✅ 已更新的密码规则

### 正则表达式
```
^[a-zA-Z0-9!@#$%^&*()_+\-=\[\]{};':"\\|,.<>\/?`~]+$
```

### 规则说明
1. **长度要求**: 8-20 个字符
2. **允许的字符**:
   - ✅ 大写字母: A-Z
   - ✅ 小写字母: a-z
   - ✅ 数字: 0-9
   - ✅ 特殊字符: `!@#$%^&*()_+-=[]{};':",./<>?\`~`

### 有效密码示例
```
✅ Abc@12345678
✅ Test@1234
✅ Password1!
✅ MyP@ssw0rd
✅ Hello#2024
✅ User_12345
✅ Admin@99
✅ Qwerty$123
```

### 无效密码示例
```
❌ 12345678          (缺少字母)
❌ abcdefgh          (缺少数字)
❌ Abc@12            (长度不足8位)
❌ Abc@12345678901234567890  (超过20位)
❌ Abc 12345         (包含空格，不允许)
❌ Abc@中文          (包含中文字符，不允许)
```

## 📝 实现位置

文件: [`lib/utils/validator.dart`](d:\project\bookkeep_flutter\flutter_application_1\lib\utils\validator.dart)

```dart
// 密码验证正则表达式
static final RegExp _passwordRegExp = RegExp(
  "^[a-zA-Z0-9!@#\$%^&*()_+\\-=\\[\\]{};':\"\\\\|,.<>\\/?`~]+\$",
);

/// 验证密码 (8-20字符,字母数字特殊字符)
static String? validatePassword(String? value) {
  if (value == null || value.isEmpty) {
    return '密码不能为空';
  }
  if (value.length < 8 || value.length > 20) {
    return '密码长度必须在8-20字符之间';
  }
  if (!_passwordRegExp.hasMatch(value)) {
    return '密码只能包含字母、数字和特殊字符';
  }
  return null;
}
```

## 🔒 安全建议

虽然当前规则允许很多特殊字符，但建议：
1. 至少包含一个大写字母
2. 至少包含一个小写字母
3. 至少包含一个数字
4. 至少包含一个特殊字符
5. 不使用常见密码（如 password123）

如需更严格的验证，可以添加额外的检查逻辑。
