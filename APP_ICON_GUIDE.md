# 应用图标设计指南

## 📱 记账主题图标设计方案

### 设计理念
体现"智能记账、理财助手"的核心定位，采用现代简约风格，紫色渐变配色。

### 视觉元素
- **主体**：钱包/账本图标
- **装饰**：货币符号（¥）徽章
- **配色**：紫色渐变 #6366F1 → #8B5CF6
- **风格**：圆角、扁平化、高识别度

### 图标规格
- **尺寸**：1024x1024 像素
- **格式**：PNG（支持透明背景）
- **安全区**：中心80%区域

## 🎨 设计方案

### 方案A：紫色渐变钱包（推荐）
```
┌─────────────────────┐
│                     │
│   ╭───────────╮    │
│   │  💼 钱包  │    │
│   ╰───────────╯    │
│         ¥          │
│                     │
└─────────────────────┘
```

**特点**：
- 圆角矩形背景（紫色渐变）
- 白色钱包图标居中
- 右下角¥符号徽章

### 方案B：账本+图表
```
┌─────────────────────┐
│                     │
│   📊 账本图标      │
│   ═══              │
│   ═══              │
│                     │
└─────────────────────┘
```

**特点**：
- 打开的账本造型
- 内含简单柱状图元素
- 体现数据分析能力

### 方案C：极简钱包轮廓
```
┌─────────────────────┐
│                     │
│    ╭──────╮        │
│    │ 👛   │        │
│    ╰──────╯        │
│                     │
└─────────────────────┘
```

**特点**：
- 线条简洁
- 单色或双色设计
- 适合深色模式

## 🛠️ 生成步骤

### 1. 准备源文件
使用以下工具之一设计图标：
- **在线工具**：Figma, Canva
- **专业软件**：Adobe Illustrator, Sketch
- **免费工具**：GIMP, Inkscape

### 2. 导出PNG
- 尺寸：1024x1024
- 格式：PNG-24（支持透明）
- 文件名：`app_icon.png`

### 3. 放置文件
将图标放入 `assets/icon/app_icon.png`

### 4. 自动生成
```bash
flutter pub get
dart run flutter_launcher_icons
```
